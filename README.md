# TenTransPro Full

API dịch tài liệu đa định dạng:
- Word (.docx)
- Excel (.xlsx)
- PDF (.pdf)
- Ảnh (.png, .jpg, .bmp)

Trích xuất nội dung → Dịch song ngữ bằng GPT → Trả về file Word/Excel song ngữ.

## Yêu cầu hệ thống
- Python 3.9+
- Cài tesseract-ocr nếu dùng OCR ảnh:
  sudo apt install tesseract-ocr
